## Submission text (optional)

Three-container Docker stack that turns any Pi, Mac mini, or Linux box into a Time Machine target for every Mac on your LAN. Samba 4.18 + vfs_fruit for the protocol, Avahi for Bonjour discovery, FastAPI + Jinja + htmx for a small admin UI. Multi-arch images for amd64 / arm64 / armv7. MIT.

Author comment should cover: the gap left by Time Capsule discontinuation and macOS removing AFP, the vfs_fruit parameters that matter, and the per-user share strategy for multi-Mac households.
